# Vantura
Vantura adalah platform kecerdasan finansial untuk merencanakan masa pensiun. Dengan memadukan profil risiko pribadi, simulasi kondisi darurat, dan data ekonomi terkini, Vantura memproyeksikan kecukupan tabungan Anda serta memberikan langkah konkret agar Anda terhindar dari krisis keuangan di masa tua.
